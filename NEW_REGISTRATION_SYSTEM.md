## 👨‍👩‍👧 ربط ولي الأمر عبر رمز الدعوة و QR

### الميزة الجديدة الكاملة:

#### **الخطوة 3: رمز دعوة الطالب** (لولي الأمر فقط)

```
┌────────────────────────────────────┐
│  🔗 ربط حسابك بالطالب             │
│  أدخل رمز الدعوة من الطالب أو      │
│  امسح رمز QR                       │
├────────────────────────────────────┤
│                                    │
│  🔲 رمز دعوة الطالب                │
│  ┌──────────────────────────┐     │
│  │    ABC123XY              │     │
│  └──────────────────────────┘     │
│                                    │
│  [✓ التحقق من الرمز]               │
│                                    │
├────────────────────────────────────┤
│  📷 يمكنك أيضاً مسح رمز QR         │
│  [مسح رمز QR]                      │
├────────────────────────────────────┤
│  ✓ تم التحقق بنجاح!                │
│  الطالب: أحمد محمد                 │
└────────────────────────────────────┘
```

### كيف يعمل:

1. **ولي الأمر يدخل في مرحلة التسجيل:**
   - يختار "ولي أمر" ك نوع الحساب
   - يدخل البيانات الشخصية
   - **يظهر له خطوة جديدة: رمز دعوة الطالب**

2. **إدخال رمز الدعوة:**
   - يكتب الرمز المكون من 8 أحرف
   - يضغط "التحقق من الرمز"
   - النظام يتحقق من الرمز في Firebase
   - إذا صحيح → يظهر معلومات الطالب
   - إذا خاطئ → يظهر رسالة خطأ

3. **مسح رمز QR (مستقبلاً):**
   - يمكن إضافة مكتبة QR scanning
   - حالياً يوجد placeholder جاهز

4. **بعد التحقق:**
   - يتم حفظ `parentCode` و `studentInfo`
   - يمكن المتابعة للتسجيل
   - عند اكتمال التسجيل، يتم إنشاء طلب ربط تلقائياً

## 🎯 سير العمل الكامل

### تسجيل ولي الأمر مع ربط الطالب:

```
1. ولي الأمر يضغط "إنشاء حساب جديد"
   ↓
2. يختار "ولي أمر"
   ↓
3. يدخل البيانات الشخصية:
   - الاسم الكامل
   - البريد الإلكتروني
   - رقم الهاتف
   ↓
4. يدخل رمز دعوة الطالب:
   - يكتب الرمز (مثلاً: ABC123XY)
   - يضغط "التحقق"
   - يظهر: "الطالب: أحمد محمد - الصف الخامس"
   ↓
5. التحقق من الهاتف:
   - يدخل رمز OTP
   ↓
6. كلمة المرور:
   - يدخل كلمة مرور قوية
   ↓
7. إنشاء الحساب:
   - يتم إنشاء حساب ولي الأمر
   - يتم إرسال طلب ربط للطالب
   - يظهر رسالة نجاح
   ↓
8. الطالب يوافق على الطلب
   ↓
9. ولي الأمر يرفع وثيقة الإثبات
   ↓
10. الطالب يراجع الوثيقة
    ↓
11. الإدارة توافقق على الطلب
    ↓
✅ ولي الأمر مرتبط بالطالب!
```

## 🚀 كيفية الاستخدام

### للمستخدمين:

1. **فتح صفحة التسجيل:**
   ```
   http://localhost:5173/register
   أو من صفحة تسجيل الدخول اضغط "إنشاء حساب جديد"
   ```

2. **اختيار نوع الحساب:**
   - طالب 🎓
   - معلم 📚
   - ولي أمر 👨‍👩‍👧
   - مشرف 🛡️

3. **اتباع الخطوات:**
   - شريط التقدم يوضح في أي خطوة أنت
   - كل خطوة لها زر "التالي" أو "السابق"

4. **لولي الأمر:**
   - في خطوة "رمز دعوة الطالب"
   - أدخل الرمز من الطالب
   - اضغط "التحقق من الرمز"
   - إذا صحيح، تابع للتسجيل

### للمطورين:

```typescript
// في RegisterPage.tsx:

// التحقق من رمز دعوة ولي الأمر
const handleValidateParentCode = async () => {
  const result = await validateParentInviteCode(parentCode);
  if (result.valid) {
    setParentCodeValid(true);
    setStudentInfo(result.student);
  }
};

// إنشاء حساب ولي الأمر مع طلب ربط
const handleSubmit = async () => {
  // ... إنشاء الحساب
  
  // إذا كان ولي أمر وأدخل رمز صحيح
  if (userType === 'parent' && parentCodeValid && studentInfo) {
    // إنشاء طلب ربط تلقائياً
    await createParentLinkRequest(
      parentUid, studentInfo.uid, parentCode
    );
  }
};
```

## 🐛 حل المشاكل الشائعة

### المشكلة: رمز الدعوة غير صالح
**الحل:**
1. تأكد من أن الطالب أنشأ رمز الدعوة
2. الرمز يجب أن يكون 8 أحرف
3. الرمز صالح لمدة 7 أيام فقط
4. اطلب من الطالب إنشاء رمز جديد

### المشكلة: لا تظهر خطوة رمز الدعوة
**الحل:**
1. تأكد من اختيار "ولي أمر" ك نوع الحساب
2. أكمل خطوة البيانات الشخصية أولاً
3. خطوة رمز الدعوة تأتي بعد البيانات الشخصية

### المشكلة: التحقق من الهاتف لا يعمل
**الحل:**
1. تأكد من إدخال رقم هاتف صحيح
2. اضغط زر "تأكيد" لإرسال OTP
3. الرمز التجريبي هو: `12345`
