## 🚀 خطوات النشر

### 1. التثبيت
```bash
npm install
```

### 2. التكوين
```bash
cp .env.example .env
# عدل .env وأضف بيانات Firebase
```

### 3. الاختبار
```bash
npm run test:run
npm run test:coverage
```

### 4. البناء
```bash
npm run build:prod
```

### 5. ترحيل كلمات المرور
```bash
npm run migrate:passwords
```

### 6. نشر قواعد الأمان
```bash
firebase deploy --only database:rules
```

### 7. النشر النهائي
```bash
firebase deploy --only hosting
```
