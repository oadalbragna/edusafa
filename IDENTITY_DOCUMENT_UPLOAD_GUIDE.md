# نظام رفع مستندات الهوية عبر جسر تيليغرام

## نظرة عامة

تم إنشاء نظام جديد لرفع مستندات إثبات الهوية (بطاقة الهوية، جواز السفر، وغيرها) عبر جسر تيليغرام بشكل آمن وموثوق.

## الملفات المُنشأة

### 1. مكون رفع مستندات الهوية
**الموقع:** `components/parent/IdentityDocumentUpload.tsx`

مكون تفاعلي يسمح لولي الأمر برفع مستندات إثبات الهوية:
- بطاقة الهوية الوطنية
- جواز السفر
- دفتر العائلة
- شهادة الميلاد
- رخصة القيادة
- تصريح الإقامة
- مستندات أخرى

**المميزات:**
- ✅ رفع الملفات عبر جسر تيليغرام
- ✅ معاينة الصور قبل الرفع
- ✅ سحب وإفلات الملفات
- ✅ التحقق من نوع وحجم الملف (حد أقصى 10 ميجابايت)
- ✅ اختيار نوع المستند بواجهة بطاقات مرئية
- ✅ شريط تقدم الرفع
- ✅ دعم الوضع الداكن

### 2. خدمة إدارة مستندات الهوية
**الموقع:** `services/identityDocument.service.ts`

خدمة Firebase لإدارة مستندات الهوية:

**الدوال المتوفرة:**
- `saveParentIdentityDocument()` - حفظ مستند هوية جديد
- `getParentIdentityDocuments()` - جلب جميع مستندات الهوية
- `updateIdentityDocumentStatus()` - تحديث حالة المستند (موافقة/رفض)
- `getLatestApprovedIdentityDocument()` - جلب آخر مستند موافق عليه
- `hasPendingIdentityDocument()` - التحقق من وجود مستند قيد المراجعة

**هيكل التخزين:**
```
sys/users/{parentUid}/identityDocuments/
  └─ {documentId}
      ├─ id: string
      ├─ documentType: string
      ├─ fileUrl: string (Telegram Bridge URL)
      ├─ fileName: string
      ├─ fileId: string (Telegram file ID)
      ├─ shortId: string
      ├─ uploadedAt: string
      └─ status: 'pending' | 'approved' | 'rejected'
```

### 3. صفحة مراجعة المستندات (للإدارة)
**الموقع:** `pages/Parent/IdentityDocumentReviewPage.tsx`

صفحة للإدارة لمراجعة مستندات الهوية المرفوعة:

**المميزات:**
- ✅ عرض بيانات ولي الأمر
- ✅ معاينة المستند (صور وPDF)
- ✅ الموافقة أو رفض المستند
- ✅ إضافة ملاحظات المراجعة
- ✅ نموذج تأكيد قبل المعالجة

**المسار:**
```
/admin/identity-documents/review/:parentUid/:documentId
```

### 4. تحديث صفحة قبول ولي الأمر
**الموقع:** `pages/Parent/ParentAcceptancePage.tsx`

تم دمج مكون رفع مستندات الهوية في صفحة قبول ولي الأمر:
- يظهر القسم بعد موافقة الطالب على طلب الربط
- يمكن لولي الأمر رفع مستند إثبات القرابة **و** مستند الهوية
- يتم حفظ مستند الهوية بشكل منفصل في قاعدة البيانات

### 5. تحديث أنواع البيانات
**الموقع:** `types/index.ts`

تم إضافة حقول جديدة لواجهة `ParentLinkRequest`:
```typescript
identityDocumentUrl?: string;
identityDocumentType?: 'id_card' | 'passport' | 'family_book' | ...;
identityDocumentId?: string;
identityUploadedAt?: string;
identityReviewedBy?: string;
identityReviewedAt?: string;
identityStatus?: 'pending' | 'approved' | 'rejected';
```

## كيفية الاستخدام

### لولي الأمر

1. **الدخول لصفحة الربط**
   - توجه إلى `/parent-accept`
   - أدخل كود دعوة الطالب

2. **انتظار موافقة الطالب**
   - بعد إرسال الطلب، ينتظر موافقة الطالب

3. **رفع المستندات**
   - بعد موافقة الطالب، يظهر قسمان:
     - **رفع وثيقة إثبات القرابة** (إثبات العلاقة بالطالب)
     - **رفع مستند إثبات الهوية** (بطاقة هوية، جواز سفر، إلخ)

4. **اختيار نوع المستند**
   - اختر نوع المستند من البطاقات المرئية
   - اضغط على الملف لاختياره أو اسحبه وأفلته

5. **الرفع عبر تيليغرام**
   - يتم رفع الملف تلقائياً عبر جسر تيليغرام
   - يتم حفظ الرابط الآمن في قاعدة البيانات

### للإدارة

1. **مراجعة المستندات**
   - توجه إلى `/admin/identity-documents/review/{parentUid}/{documentId}`
   - راجع بيانات ولي الأمر والمستند المرفوع

2. **الموافقة أو الرفض**
   - أضف ملاحظاتك (اختياري)
   - اضغط على "الموافقة" أو "رفض"
   - أكد العملية

## الفروقات عن النظام السابق

| الميزة | النظام السابق (Proof Document) | النظام الجديد (Identity Document) |
|--------|-------------------------------|-----------------------------------|
| **الغرض** | إثبات القرابة بالطالب | إثبات هوية ولي الأمر |
| **التخزين** | Firebase Storage | Telegram Bridge |
| **نوع الملف** | صورة أو PDF | صورة أو PDF (حتى 10MB) |
| **المراجعة** | من الطالب ثم الإدارة | من الإدارة مباشرة |
| **التخزين في DB** | في طلب الربط | في مسار منفصل للمستخدم |

## أنواع المستندات المدعومة

| النوع | الوصف | الحد الأقصى |
|-------|-------|-------------|
| بطاقة الهوية | البطاقة الوطنية أو الشخصية | 10 ميجابايت |
| جواز السفر | جواز سفر ساري المفعول | 10 ميجابايت |
| دفتر العائلة | دفتر العائلة أو سجل الأسرة | 10 ميجابايت |
| شهادة الميلاد | شهادة الميلاد الرسمية | 10 ميجابايت |
| رخصة القيادة | رخصة القيادة كإثبات هوية | 10 ميجابايت |
| تصريح الإقامة | تصريح الإقامة أو الإقامة الدائمة | 10 ميجابايت |
| مستند آخر | أي مستند رسمي آخر | 10 ميجابايت |

## الأمان

- ✅ الملفات تُرفع عبر **جسر تيليغرام** المشفر
- ✅ الروابط آمنة ويمكن تمويهها إذا لزم الأمر
- ✅ المراجعة تتم من الإدارة قبل الموافقة
- ✅ حفظ جميع بيانات المراجعة (من، متى، ملاحظات)
